# Sistema de Personas y Cuentas Bancarias

Sistema básico de gestión bancaria en Java que permite crear personas, asociarlas a cuentas bancarias y realizar operaciones de depósito y retiro.

## Estructura del Proyecto

```
proyecto/
├── clases/
│   ├── Persona.java
│   └── CuentaBancaria.java
└── Aplicacion.java
```

## Características

### Clase Persona
- Almacena nombre y edad
- Métodos getters/setters
- Método para mostrar información

### Clase CuentaBancaria
- Gestiona saldo y titular (Persona)
- Número de cuenta generado automáticamente (6 dígitos)
- Lista de todas las cuentas creadas
- Operaciones: depositar, retirar, consultar información



## Objetivos de Aprendizaje

- Encapsulamiento con atributos privados
- Constructores e instanciación
- Gestión de colecciones (ArrayList)
- Atributos y métodos estáticos
- Relaciones entre clases

---

**Nota:** Este es un proyecto educativo para practicar conceptos de Programación Orientada a Objetos en Java.
